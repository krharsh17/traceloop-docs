---
title: "Status of user deletion request"
api: "GET https://app.traceloop.com/api/config/privacy/data-deletion"
---

You can view the status of your data deletion requests using this endpoint.

## Request Query Parameter

<ParamField query="requestId" type="string">
  The request ID received after creating a [user deletion request](https://www.traceloop.com/docs/api-reference/privacy/delete_request).
</ParamField>

## Response

<ResponseField name="completed" type="boolean">
  `true` if the process was completed, `false` otherwise.
</ResponseField>

<ResponseField name="deleted" type="string">
  The number of spans that have been deleted.
</ResponseField>

<ResponseField name="total" type="string">
  The total number of spans that were to be deleted.
</ResponseField>
