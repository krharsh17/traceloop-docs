The following is a list of publicly available APIs you can use with the [Traceloop platform](https://app.traceloop.com).

- Tracing
    - [Logging Prompts and Responses](https://www.traceloop.com/docs/api-reference/tracing/whitelist_user)
- GDPR & Privacy
    - [Delete user specific data](https://www.traceloop.com/docs/api-reference/privacy/delete_request)
    - [View status of user data deletion request](https://www.traceloop.com/docs/api-reference/privacy/delete_status)

All APIs require an API key to be used. You can get one by [signing up](https://app.traceloop.com),
and then going to the [API Keys](https://app.traceloop.com/settings/api-keys) page.
